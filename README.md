# 🎓 Academia Barro Branco - Operação Cadete

> Plataforma educacional moderna para preparação ao concurso de Aluno-Oficial da Polícia Militar do Estado de São Paulo.

## 🚀 Visão Geral

Academia Barro Branco é uma plataforma de estudos profissional, rápida e responsiva, desenvolvida com HTML5, CSS3 e JavaScript puro.

### ✨ Características

- ✅ **Design Moderno** - Inspirado em Notion, Duolingo e Apple
- ✅ **Mobile First** - Funcionará perfeitamente em qualquer dispositivo
- ✅ **Modo Escuro** - Tema claro e escuro integrado
- ✅ **PWA** - Funciona offline com Service Worker
- ✅ **Sem Frameworks** - HTML5, CSS3 e JavaScript puro
- ✅ **Código Limpo** - Bem organizado e documentado
- ✅ **GitHub Pages Ready** - Deploy automático

## 📁 Estrutura do Projeto

```
academia-barro-branco/
├── index.html                 # Página inicial
├── manifest.json              # Configuração PWA
├── service-worker.js          # Service Worker (offline)
├── css/
│   ├── style.css             # Estilos globais
│   └── dashboard.css         # Estilos do dashboard
├── js/
│   ├── app.js                # Aplicação principal
│   ├── countdown.js          # Contagem regressiva
│   └── navigation.js         # Navegação
├── pages/
│   ├── dashboard.html        # Dashboard
│   ├── cronograma.html       # Cronograma
│   ├── materias.html         # Matérias
│   └── perfil.html           # Perfil
├── assets/                    # Imagens e ícones
├── data/                      # Dados JSON
├── .gitignore
└── README.md
```

## 🛠 Tecnologias

- **HTML5** - Semântica moderna
- **CSS3** - Design responsivo com variáveis
- **JavaScript ES6+** - Sem dependências externas
- **PWA** - Progressive Web App
- **LocalStorage** - Persistência de dados

## 🎨 Design System

- Paleta de cores inspirada em Notion
- Tipografia moderna com Google Fonts
- Componentes reutilizáveis
- Dark mode automático
- Animações suaves

## 📱 Responsividade

- Mobile First
- Breakpoints: 480px, 768px, 1024px, 1440px
- Touch-friendly
- Otimizado para a web

## 🚀 Como Iniciar

1. Clone o repositório
```bash
git clone https://github.com/thomaztrade/academia-barro-branco.git
```

2. Abra `index.html` no navegador

3. Instale como PWA (clique no ícone de instalação)
